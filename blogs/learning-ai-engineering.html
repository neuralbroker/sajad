Learning AI Engineering as a Software Developer

Learning · Jun 2025 · ~10 min read

My journey from traditional SWE to AI engineering. What worked, what didn't.

Why AI Engineering?

As a software developer, I saw AI transforming how we build software. I wanted to be part of this shift.

How I Started

  • Started with OpenAI API tutorials
  • Built small projects with GPT-3
  • Learned prompt engineering basics

Challenges

  • Different mental model: Not traditional programming
  • Uncertainty: Non-deterministic outputs
  • Evaluation: Hard to measure quality

What Worked

  • Building projects: Start small, iterate fast
  • Reading papers: Stay updated on latest research
  • Community: Join AI developer communities

Key Frameworks

  • LangChain: Most popular for building LLM apps
  • LangGraph: For complex agent workflows
  • LlamaIndex: For RAG applications

Advice for Developers

  • Start with API integrations, not model training
  • Learn prompt engineering - it's a skill
  • Focus on practical applications
  • Build real projects and share them

Conclusion

The best time to start was yesterday. The second best time is now. Start building!
