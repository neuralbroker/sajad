Learning AI Engineering as a Software Developer

LearningDec 2025

My journey from traditional SWE to AI engineering. What worked, what didn't.

Where I Started

Background in Java, Python, web development. Traditional backend focus.

What Worked

  • Building projects - Actual hands-on experience beats passive learning
  • Focusing on engineering - Prompt engineering, API integration, not ML theory
  • Start simple - Chatbots → Agents → Complex systems

What Didn't

  • Too much theory - Don't need to understand transformers deeply to use LLMs
  • Chasing every new tool - Master fundamentals first

Advice

Start building. Use APIs. Learn by doing. The barrier to entry is lower than you think.
